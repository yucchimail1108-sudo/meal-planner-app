うちごはんプランナー

献立作成から買い物、食材管理までを一元化する料理管理アプリ

ホーム画面

概要

うちごはんプランナーは、献立作成から買い物、食材管理までを一元的に行える家庭向け料理管理アプリです

献立を作成することで必要な食材を自動で抽出し、不足分を買い物リストとして一覧化できます

おうち食材を登録することで在庫を考慮した買い物ができ、重複購入やフードロスを防ぐことができます

作成の目的

食事に関する家事負担を軽減し、献立作成・買い物・食材管理の効率化を目的として開発しました

重複購入やフードロスを減らし、家計や環境への負担軽減を目指しています

主な機能

  • ユーザー登録 / ログイン / ログアウト
  • レシピの登録 / 編集 / 削除
  • 献立の作成・カレンダー管理
  • 買い物リストの自動生成
  • おうち食材の管理

使用技術

  • Python / Django
  • HTML / CSS / JavaScript
  • SQLite
  • Git / GitHub

工夫した点

  • 献立 → 買い物 → 食材管理の流れを一体化
  • 在庫考慮による重複購入防止
  • 分量換算の簡単操作化
  • 外食・惣菜対応の柔軟設計
  • CSS共通化によるUI統一

今後の課題

  • 食材の数量管理対応
  • UI/UX改善
  • レシピ登録の簡略化
  • モバイル操作性向上
